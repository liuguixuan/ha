1、打开Git-bash 2、mkdir 一个文件夹 cd 这个文件夹 创建文件夹 并进入文件夹 3、Git init 初始化仓库 Git工作流 工作区：git add 从工作去到暂存区 不能包含一个空文件夹，必须包含文件 暂存区：git commit -m注释 从暂存区提交到历史区 减少提交版本的数量，方便管理 历史区 touch 1.txt 创建一个名字为1.txt的文件 git add 1.txt 吧1.txt填到暂存区 git commit -m"注释" 吧1.txt提交到历史区 git log 查看历史记录 注册账号 组长fork讲师的项目 把自己的项目下载到本地 可选git clone 仓库地址 进入这个目录 git push origin master 提交到远程仓库 推送到GitHub服务器上 git remote -v 查看远程仓库的地址
